## Getting Started

This is a standard Python project that runs on the CLI.

### Prerequisites

- python >= 3.10 (it will probably work with lower versions, but I haven't tested it)
- poetry >= 1.2

### Installation

1. Install dependencies `> poetry install`
1. Run with `> make all`
